## 🚀 使用说明

### 快速开始

```bash
# 1. 确保设备已连接（启用 USB 调试）
adb devices

# 2. 配置 Web AI
npm run dev webai add doubao https://www.doubao.com/chat/

# 3. 启动聊天
npm run dev chat --webai doubao

# 4. 发送指令
刷抖音
```

### 支持的指令

```
刷抖音                        # 浏览 10 个视频
刷 20 个视频                  # 浏览 20 个视频
刷抖音，点赞                  # 浏览并点赞
刷抖音，点赞，评论            # 浏览、点赞、评论
刷 15 个视频并进行分析        # 浏览并 AI 分析
```
